2. Встав правильну форму

Дивись на рід: Milan = mužský, Martina = ženský, izba = ženský rod.

1. Milan je naj______ žiak z triedy. / vysoký /
2. Peter je ______ ako Milan. / vysoký /
3. Martina je ______ ako Peter. / nízka /
4. Táto izba je ______ ako moja izba. / veľká /
5. Moja obývačka je ______ než študentská izba. / pohodlná /
6. Tento obraz je ______ zo všetkých. / pekný /
1. najvyšší · 2. vyšší · 3. nižšia · 4. väčšia · 5. pohodlnejšia · 6. najkrajší

3. Вибери правильну відповідь

Тут тільки одна правильна відповідь.

1. Robert má 49 rokov. Karol má 27 rokov. Robert je...
2. Zuzana má 70 rokov, Robert 49, Karol 27. Zuzana je...
3. Moja izba je 10 m², tvoja izba je 16 m². Tvoja izba je...
4. dobrý → komparatív je...
5. zlý → superlatív je...
1. starší ako Karol · 2. najstaršia · 3. väčšia ako moja · 4. lepší · 5. najhorší

2. Моя кімната більша, ніж твоя кімната.
